# global-warming
в ходе этого проекта я реализую свою идею, направленную на остановку глобального потепления.
я собираюсь сделать телеграмм бота с следующим функционалом:

1)структурированная информация о проблеме потепления 
2)информация о способах решения этой проблемы
3)конкретные инструкции о некоторых способах решения проблемы (расчет энергоэфективности дома, сортировка мусора итд)
4) хранилище с мемами про экологию, которые будут способствовать распространению информации о данной проблеме

в перспективе:
1)сделать сайт для хранения большого кол-ва собранной информации о проблеме и угрозе глабального потепления 

совсем в перспективе:
1)при помощи ИИ генерировать мемы, а не просто собирать хранилище
2)внедрить в чат бота помошник с ИИ
