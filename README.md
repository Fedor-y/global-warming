# global-warming
проект выполнял - Федор Пучков 

задачи проекта:
распространить информацию о проблеме глобального потепления и постараться замедлить ее.

как реалезованы поставленные задачи:

в ходе своего проекта был написан телграмм бот со следующими функциями:
1)расчет энергоэффективности своего дома
2)короткие советы, которые медленно но верно помогут решить проблему с потеплением
3)справочник по сортировке мусора
4)экологичесские мемы, простые, но добавляют немного позитивной атмосферы
